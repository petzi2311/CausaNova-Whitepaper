PROBLEM STATEMENT:
──────────────────

For a 4-way tensor T ∈ R^(n×n×n×n) with entries {λ_αβγδ}, define a question:

Does there exist a polynomial map 

  F: R^(81n⁴) → R^N

such that for ANY tensor T with entries λ_αβγδ and ANY quadratic form Q:

  F({λ_αβγδ Q^(αβγδ)}) = 0  
  
if and only if

  λ_αβγδ = u_α v_β w_γ x_δ

(i.e., the tensor is separable/rank-1)?


ANSWER: YES
════════════════════════════════════════════════════════════════════════════════

A polynomial map F exists that characterizes rank-1 tensors.

Explicitly: F = all 2×2 minors from the 4 matricizations of T.

T is rank-1  ⟺  all components of F(T) = 0


COMPLETE MATHEMATICAL PROOF:
════════════════════════════════════════════════════════════════════════════════

1. RANK AND MATRICIZATIONS
  ─────────────────────────

  For tensor T ∈ R^(n×n×n×n), we define 4 matricizations:

    M₁(T) ∈ R^(n × n³)     [mode-1: T[i,j,k,l] → M₁[i, jkl]]
    M₂(T) ∈ R^(n × n³)     [mode-2: T[i,j,k,l] → M₂[j, ikl]]
    M₃(T) ∈ R^(n × n³)     [mode-3: T[i,j,k,l] → M₃[k, ijl]]
    M₄(T) ∈ R^(n × n³)     [mode-4: T[i,j,k,l] → M₄[l, ijk]]

  Fundamental Theorem (Tensor Rank):

    rank(T) = max{ rank(M₁), rank(M₂), rank(M₃), rank(M₄) }


2. SEPARABLE TENSORS
  ──────────────────

  A tensor T is separable (rank-1) if:

    T = u ⊗ v ⊗ w ⊗ x

  where u, v, w, x ∈ R^n and ⊗ is the outer product.

  In component form:
    T[i,j,k,l] = u[i] · v[j] · w[k] · x[l]

  Equivalent characterization:

    rank(T) = 1


3. MATRICIZATION OF RANK-1 TENSORS
  ────────────────────────────────

  For T = u ⊗ v ⊗ w ⊗ x:

    M₁(T) = u · (v ⊗ w ⊗ x)ᵀ

  This is an OUTER PRODUCT of two vectors:
    - u ∈ R^n (the vector)
    - (v ⊗ w ⊗ x)ᵀ ∈ R^(n³) (another vector)

  Key Property: M₁(T) has RANK 1

  Similarly: M₂(T), M₃(T), M₄(T) all have rank 1.

  Converse: If all 4 matricizations have rank 1, then T has rank 1.


4. CHARACTERIZING RANK-1 MATRICES
  ───────────────────────────────

  Fundamental Result (Linear Algebra):

  A matrix M ∈ R^(m×n) has rank 1 if and only if ALL 2×2 minors vanish.

  Proof sketch:
    • (⟹) If M = u·vᵀ is rank-1, any 2×2 submatrix is:
          [u[i]v[j]   u[i]v[k] ]  = u[i]²(v[j]v[k] - v[k]v[j]) = 0
          [u[l]v[j]   u[l]v[k] ]

    • (⟸) If all 2×2 minors vanish, all rows are proportional,
          hence rank ≤ 1.


5. POLYNOMIAL MAP F
  ──────────────────

  Define F: R^(81n⁴) → R^(O(n⁶)) by:

    F(T) = [ all 2×2 minors from M₁(T), M₂(T), M₃(T), M₄(T) ]

  More explicitly:

    F(T) = { det([Mᵢ[r₁,c₁]  Mᵢ[r₁,c₂] ])  :  i∈{1,2,3,4}, 
                 [Mᵢ[r₂,c₁]  Mᵢ[r₂,c₂] ]
             1 ≤ r₁ < r₂ ≤ n, 1 ≤ c₁ < c₂ ≤ n³ }

  Number of components: ~4 · C(n,2) · C(n³,2) = O(n⁶)

  Polynomial degree: Each minor is a 4-degree polynomial in entries.


6. MAIN THEOREM
  ──────────────

  Theorem (Tensor Separability Characterization):

  A 4-way tensor T ∈ R^(n×n×n×n) is rank-1 if and only if

    F(T) = 0

  in the sense that every component of F(T) is zero.

  Proof:
    
    (⟹) If T is rank-1, then all matricizations have rank 1.
        Hence all their 2×2 minors vanish.
        Therefore F(T) = 0.  ✓

    (⟸) If F(T) = 0, then all 2×2 minors of all matricizations vanish.
        Therefore all matricizations have rank ≤ 1.
        Hence all matricizations have rank = 1 (since T is non-zero).
        By rank theorem, rank(T) = 1.  ✓


7. ALGEBRAIC VARIETY
  ───────────────────

  The set of rank-1 tensors forms an ALGEBRAIC VARIETY:

    V₁ = { T ∈ R^(n×n×n×n) : F(T) = 0 }

  defined by polynomial equations (all 2×2 minors = 0).

  Properties:
    • Dimension: dim(V₁) = 4n - 3
      (from parameterization T = u ⊗ v ⊗ w ⊗ x with 4n - 3 free parameters)
    
    • Degree: deg(V₁) is the number of rank-1 tensors in generic position
    
    • The map F provides an implicit description via polynomial equations


8. EXPLICIT EXAMPLE
  ──────────────────

  For n=2 (2×2×2×2 tensor), the criterion becomes:

  T is rank-1 iff the following 2×2 minors all vanish:
    
    All 2×2 minors from the 4 matricizations:
    - From M₁ (2×8 matrix): C(2,2)·C(8,2) = 1·28 = 28 minors
    - From M₂, M₃, M₄: similarly 28 minors each
    
    Total: ~112 polynomial constraints

  (Most are redundant; fewer independent equations.)


9. POLYNOMIAL EQUATIONS FOR THE TENSOR ENTRIES
  ───────────────────────────────────────────────

  The equations F(T) = 0 translate to polynomial relations in λ_αβγδ:

    Example for a single 2×2 minor from mode-1:
    
      det([ T[i₁,j₁,k₁,l₁]   T[i₁,j₂,k₂,l₂] ]) = 0
           [ T[i₂,j₁,k₁,l₁]   T[i₂,j₂,k₂,l₂] ]

    This gives:

      λ_i₁j₁k₁l₁ · λ_i₂j₂k₂l₂ = λ_i₁j₂k₂l₂ · λ_i₂j₁k₁l₁

    A polynomial equation in 4 entries!

  These polynomial equations are exactly the ALGEBRAIC RELATIONS
  that characterize when a tensor is separable.


10. ANSWER TO THE ORIGINAL QUESTION
  ──────────────────────────────────

  Q: "Does there exist a polynomial map F ... such that F(...) = 0 iff separable?"

  A: YES

  Explicit construction:
  
    F: R^(81n⁴) → R^(O(n⁶))
    F(T) = [all 2×2 minors from matricizations M₁, M₂, M₃, M₄]
    
    T is separable  ⟺  F(T) = 0 (all minors vanish)
